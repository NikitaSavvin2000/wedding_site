# Руководство по конфигурации сайта

## Быстрая настройка

Все основные параметры сайта вынесены в файл `/src/app/config/theme.ts` для удобной настройки без редактирования компонентов.

## Параметры конфигурации

### 1. Цвета текста (HEX формат)

```typescript
hero: {
  namesColor: '#FEECD0',  // Цвет имен "Никита" и "Елизавета" на главной странице
  dateColor: '#FEECD0',   // Цвет даты "19 • 08 • 2026" на главной странице
}
```

**Как изменить:**
- Откройте `/src/app/config/theme.ts`
- Измените значения `namesColor` и `dateColor` на нужный HEX-цвет
- Пример: `namesColor: '#FFB6C1'` для светло-розового

---

### 2. Размеры карточек программы дня

```typescript
venueCards: {
  containerMaxWidth: '1600px',  // Максимальная ширина контейнера с карточками
  
  imageHeight: {
    mobile: '400px',   // Высота изображения на мобильных устройствах
    desktop: '500px',  // Высота изображения на десктопе
  },
  
  cardPadding: {
    mobile: 'p-10',   // Внутренние отступы карточки на мобильных
    desktop: 'p-12',  // Внутренние отступы карточки на десктопе
  },
}
```

**Как изменить размер карточек:**

Для увеличения карточек:
- `containerMaxWidth: '1800px'` - сделает карточки шире
- `imageHeight.desktop: '600px'` - увеличит высоту фотографий
- `cardPadding.desktop: 'p-16'` - увеличит внутренние отступы

Для уменьшения карточек:
- `containerMaxWidth: '1400px'` - сделает карточки уже
- `imageHeight.desktop: '400px'` - уменьшит высоту фотографий
- `cardPadding.desktop: 'p-8'` - уменьшит внутренние отступы

**Адаптивность:**
Карточки автоматически адаптируются:
- На десктопе: 3 карточки в ряд
- На планшетах и мобильных: по 1 карточке

### 3. Фильтры фотографий

```typescript
imageFilter: {
  contrast: 1.05,     // Контрастность (1.0 = оригинал, 1.2 = +20%)
  saturate: 0.75,     // Насыщенность (0.5 = -50%, 1.0 = оригинал)
  brightness: 1.08,   // Яркость (0.9 = темнее, 1.1 = светлее)
  sepia: 0.25,        // Эффект сепии (0 = нет, 1 = максимум)
  hueRotate: -5,      // Сдвиг оттенка (-10 до 10 градусов)
  
  overlayOpacity: {
    top: 0.15,        // Прозрачность верха оверлея (0-1)
    bottom: 0.2,      // Прозрачность низа оверлея (0-1)
  },
}
```

**Примеры настроек фильтра:**

**Более яркие и насыщенные фото:**
```typescript
contrast: 1.1,
saturate: 0.9,
brightness: 1.15,
sepia: 0.15,
```

**Более винтажный стиль:**
```typescript
contrast: 1.0,
saturate: 0.6,
brightness: 1.05,
sepia: 0.4,
```

**Минимальный фильтр (почти оригинал):**
```typescript
contrast: 1.0,
saturate: 0.95,
brightness: 1.0,
sepia: 0.1,
hueRotate: 0,
```

**Усиленный пастельный эффект:**
```typescript
contrast: 0.95,
saturate: 0.7,
brightness: 1.12,
sepia: 0.3,
hueRotate: -8,
```

## Примеры изменений

### Пример 1: Изменить цвет имен на розовый

```typescript
hero: {
  namesColor: '#FFB6C1',  // Светло-розовый
  dateColor: '#FFB6C1',
}
```

### Пример 2: Сделать карточки больше

```typescript
venueCards: {
  containerMaxWidth: '1800px',
  imageHeight: {
    mobile: '450px',
    desktop: '600px',
  },
  cardPadding: {
    mobile: 'p-12',
    desktop: 'p-16',
  },
}
```

### Пример 3: Убрать фильтр с фотографий

```typescript
imageFilter: {
  contrast: 1.0,
  saturate: 1.0,
  brightness: 1.0,
  sepia: 0,
  hueRotate: 0,
  overlayOpacity: {
    top: 0,
    bottom: 0,
  },
}
```

---

## После изменений

1. Сохраните файл `/src/app/config/theme.ts`
2. Если используете Docker, пересоберите контейнер:
   ```bash
   docker-compose up -d --build
   ```
3. Если запускаете локально, перезагрузите страницу в браузере

## Дополнительные настройки

### Изменить пути к фотографиям
Откройте `/src/app/config/images.ts`

### Изменить текст или расписание
Откройте соответствующие компоненты в `/src/app/components/`
